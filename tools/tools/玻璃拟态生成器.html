📖 使用说明
🎨
快速预设：一键切换极光紫、深海蓝、日落橙等场景配色，快速上手。
🔮
光球背景：三个彩色光球构成渐变背景，是玻璃拟态效果的核心视觉来源。
💨
模糊强度：控制 backdrop-filter blur 值，越高玻璃感越强，建议 8–24px。
🪟
透明度：玻璃背景的白色（或色调）填充不透明度，越低越透明。
✏️
玻璃色调：改变玻璃层的底色，可快速切换为带色彩的彩色玻璃。
📐
边框：半透明边框是玻璃效果的关键细节，可调节宽度和不透明度。
📋
多格式导出：支持复制标准 CSS、Tailwind 内联样式 和 CSS 变量三种格式。
