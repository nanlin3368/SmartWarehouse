📖 使用说明
📁
上传图片：拖放或点击选择需要加水印的图片，支持 JPG / PNG / WEBP / BMP
✏️
文字水印：自定义文字内容、字号、颜色、透明度、旋转、描边、阴影、斜体等
🏷️
图片水印：上传 Logo 或签名图，支持缩放 / 透明度 / 旋转，推荐使用透明背景 PNG
📍
九宫格定位：点击九宫格快速选择位置，再用 X/Y 微调做精确偏移
🔄
平铺模式：将水印重复铺满整张图，可调节平铺间距和密度
🔒
数据安全：所有处理在浏览器本地完成，图片不上传任何服务器
