📖 使用说明
📂
导入文件：点击输入框右上角 📂 按钮直接导入任意代码文件，支持所有文本格式
🔀
对比逻辑：基于 Myers 差分算法逐行对比，绿色为 B 新增，红色为 A 删除，同时支持字符级内联高亮
📄
统一 / 分栏：统一视图上下混排差异行；分栏视图左右对照，适合宽屏阅读
⬆️
逐差导航：▲ / ▼ 按钮在各处差异之间跳转，当前差异自动高亮并滚动到视图中央
⚙️
对比选项：可忽略空白差异（缩进变化）、忽略大小写、隐藏相同行，专注查看差异内容
🔄
快速交换：点击"交换 A / B"可一键互换两侧内容，方便反向对比
📥
导出 Patch：以标准 unified diff 格式导出 .patch 文件，可直接用于 Git 或 patch 命令
🔒
数据安全：所有对比均在浏览器本地完成，代码不会上传至任何服务器
