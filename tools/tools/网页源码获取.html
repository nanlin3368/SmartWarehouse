📖 使用说明
🔗
输入网址：完整URL（含 http:// 或 https://），按 Enter 或点击按钮获取。
🔄
CORS代理：默认关闭，若遇到跨域错误可手动开启（基于 allorigins）。
🔍
关键词搜索：获取源码后可在源码内搜索关键词，支持上下导航高亮。
🗑️
清空全部：一键清空网址和源码显示区域。
🔗
提取链接：从已获取的源码中提取所有超链接，支持一键复制全部链接。
📋
复制源码：一键复制全部源代码到剪贴板。
💾
下载文件：将源码保存为以目标域名命名的 .html 文件。
🔒
数据安全：所有处理在浏览器本地完成，不收集任何信息。
