📖 使用说明
📂
导入文件：点击输入框右上角 📂 按钮，可直接从本地导入 .json 文件，自动填入对应输入框
📋
复制内容：点击 📋 按钮可将对应输入框的 JSON 内容复制到剪贴板；对比完成后点击"复制结果"可导出纯文本差异报告
📝
输入 JSON：在左侧粘贴原始 JSON（A），右侧粘贴新版本 JSON（B），支持任意嵌套结构
🔍
开始对比：点击对比按钮，工具会递归遍历所有字段，逐层列出新增、删除、修改的差异
🎨
差异高亮：绿色 表示新增字段，红色 表示删除字段，黄色 表示值被修改
🌲
折叠节点：对象和数组节点可点击 ▶ / ▼ 按钮展开或折叠，便于聚焦查看关键差异
✨
格式化：点击"格式化 A & B"可自动美化输入框中的 JSON，同时验证语法合法性
📄
格式化预览：切换到"格式化预览"模式，可查看带语法高亮的完整 JSON 内容（含 key、字符串、数字、布尔值等色彩区分）
🔒
数据安全：所有处理均在浏览器本地完成，JSON 数据不会上传至任何服务器
